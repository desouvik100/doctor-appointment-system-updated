### Spacing
- Section Gap: 40px
- Card Padding: 24px
- Element Gap: 12px-16px
- Container Padding: 40px (desktop), 20px (tablet), 16px (mobile)

### Border Radius
- Cards: 20-24px
- Buttons: 10-12px
- Inputs: 12px
- Badges: 20px (pill)

### Shadows

- Medium: 0 8px 32px rgba(0, 0, 0, 0.08)
- Strong: 0 16px 40px rgba(0, 0, 0, 0.12)

### Responsive Breakpoints
- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: 480px - 768px
- Small Mobile: < 480px
