## Troubleshooting

### Cannot Join Consultation

**Problem**: "Join Consultation" button not appearing

**Solutions**:
- Check if appointment is confirmed
- Verify payment completed
- Wait for 15-minute window before appointment
- Refresh the page

### Meeting Link Not Working

**Problem**: Cannot access meeting link

**Solutions**:
- Check internet connection
- Try different browser
- Clear browser cache
- Contact support with appointment ID

### Join Code Invalid

**Problem**: "Invalid join code" error

**Solutions**:
- Copy code exactly as shown
- Check for extra spaces
- Use code from dashboard
- Contact support if issue persists
